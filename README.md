GoalPortal Frontend

A modern enterprise KPI & Goal Management Portal frontend built using React, Vite, Tailwind CSS, and Framer Motion.

This application provides role-based dashboards for:

Employee,
Manager,
Admin / HR

with enterprise-grade UI, analytics, goal tracking, and performance management workflows.

Installation:

Clone repository:

git clone https://github.com/SuyashRSinha/goalportal-frontend.git

Move into project:

cd goalportal-frontend

Install dependencies:

npm install

Run locally:

npm run dev
Environment Variables

Create .env file:

VITE_API_URL=https://web-production-0f7c0.up.railway.app,

Build for Production
npm run build

Deployment
Frontend Deployed using: Vercel
Backend Deployed using: Railway
